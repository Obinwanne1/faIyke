# SKILL: shopify-builder

## TRIGGER
"build shopify", "create store", "design product page", "ecommerce setup"

## GOAL
Generate production-ready Shopify store structure and components.

## PROCESS
1. Identify business type and product
2. Define store structure
3. Generate key sections (homepage, PDP, collections)
4. Apply UX + conversion best practices

## OUTPUT FORMAT
- **Store Structure** — page hierarchy
- **Key Pages** — purpose + content blocks
- **Section Breakdown** — components per page
- **Conversion Features** — trust signals, CTAs, UX wins

## RULES
- Clarity and conversion over aesthetics
- No generic templates
- Output must be actionable immediately
