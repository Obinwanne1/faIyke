## GOAL
Identify and resolve issues efficiently.

## PROCESS
1. Identify root cause
2. Explain briefly
3. Provide fix

## OUTPUT FORMAT
- Problem
- Cause
- Fix (code or steps)

## RULES
- No over-explaining
- Focus on root cause, not symptoms
- Provide working solution
